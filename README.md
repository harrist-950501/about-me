# Personal Website

A small two-page personal website built with HTML and CSS.

## Overview

This project introduces me, my interests, coursework, and student involvement at the University of Washington. It also includes a second page about the International Undergraduate Student Ministry, with a shared stylesheet to keep the site visually consistent.

## Features

- Multi-page static website
- Shared CSS styling across pages
- Semantic HTML structure
- Internal and external navigation
- Organized image assets

## Built With

- HTML5
- CSS3
- Google Fonts

## Project Structure

- `about.html` - main personal introduction page
- `iusm.html` - student organization page
- `styles.css` - shared stylesheet
- `pictures/` - image assets used by the site

## Purpose

I built this project to practice writing structured HTML, reusable CSS, and consistent styling across multiple pages.
