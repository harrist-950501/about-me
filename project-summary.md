## Overview of Project

This project is a small static website built with HTML and CSS to present personal information and campus involvement. It contains two linked pages, a shared stylesheet for visual consistency, and organized image assets for each section of the site. The first page serves as an introduction to the author, including coursework, interests, clubs, and contact information. The second page highlights the International Undergraduate Student Ministry through structured content, images, and supporting links.
